# TaskMate

A simple To-Do List web app created to practice and improve my HTML, CSS, and JavaScript skills.

## Live Demo

https://mytaskmate01.netlify.app/

## Features

- Add new tasks
- Delete tasks
- Clean and simple user interface
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Clone this repository.
2. Open `index.html` in your browser.
3. Start adding your tasks.
